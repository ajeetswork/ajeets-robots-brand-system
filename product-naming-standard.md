# Ajeets Robots ‚Äì Product Naming Standard

## Approved Title Structure

```
Ajeets Robots - [Product Name]
```

- **Brand prefix:** `Ajeets Robots` (exact spelling, Title Case)
- **Separator:** ` - ` (space, hyphen, space)
- **Product name:** Title Case, specific product type (no category-level vagueness)

---

## Capitalization Rules

1. **Brand name** ‚Äî always `Ajeets Robots` (Title Case, plural "Robots", no apostrophes)
   - ‚ùå `AJEETS ROBOTS` / `ajeets robots` / `Ajeet Robot` / `Ajeets Robot's`
2. **Product name** ‚Äî Title Case for every word
   - ‚úÖ `Shower Curtain`, `Mens Tee`, `Womens Sweatshirt`
   - ‚ùå `mens tee`, `journal`
3. **Acronyms / years** ‚Äî keep as-is (`2026 Wall Calendar`)

---

## Punctuation Rules

| Rule | ‚úÖ Correct | ‚ùå Incorrect |
|------|-----------|-------------|
| Separator is exactly ` - ` (space-dash-space) | `Ajeets Robots - Mug` | `Ajeets Robots \|\| Backpack` |
| No apostrophes in the brand name | `Ajeets Robots - Polo Tee` | `Ajeets Robot's - Polo Tee` |
| No quotation marks in titles | `Ajeets Robots - Cork Coaster Set` | `‚Ä¶"ajeets robots" Logo‚Ä¶` |
| No hashtag / numbering suffixes | `Ajeets Robots - Womens Sweatshirt` | `‚Ä¶Sweatshirt #1` |
| No em dashes / en dashes as separators | `Ajeets Robots - Cork Coaster Set` | `Cork Coaster Set ‚Äî Minimal‚Ä¶` |
| No trailing whitespace in titles | `Ajeets Robots - Shower Curtain` | `Ajeets Robots - Shower Curtain‚ê£` |
| Product names use plain Title Case ‚Äî no extra symbols | `Ajeets Robots - Backpack` | `Ajeets Robots \|\| Classic Backpack` |

Additional conventions:
- **Drop the "Classic" label** ‚Äî it was applied inconsistently across 7 products and has no meaningful distinction. Use the plain product name.
- **Remove possessive apostrophes** from "Women's / Men's" ‚Üí `Womens` / `Mens` (keeps titles cleaner in URLs and catalogs)
- **One product type per title** ‚Äî no repeating the product name twice (e.g. "Coaster Set ‚Ä¶ Coasters")
- **Specific product names only** ‚Äî no category-level vagueness like "Streetwear"
